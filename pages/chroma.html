Custom SkyHanni Chroma

Max Saturation: Disabled

Instructions:

  1. Add Colors: Enter a hex color (e.g., #abcdef) and click "Add Color" to add it to the list.
  2. Max Saturation:Click to toggle max saturation. When enabled, colors will appear more vibrant.
  3. Set Interpolation: Enter a number to adjust how smoothly colors transition. Higher values = smoother.
  4. Generate Shaders: After configuring, click "Generate Shaders" to create and download the shader files.
